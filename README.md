# La Salina Reserve - Hacienda Digital

Portal interactivo de monitoreo forestal, adquisición de árboles y membresías para la reserva agroforestal **La Salina Reserve**, ubicada en El Cerrito, Valle del Cauca, Colombia.

## 🚀 Características
- **Hacienda Virtual:** Mapa interactivo procedural de 20 hectáreas con 10,000 árboles.
- **Monitoreo IoT:** Simulación de telemetría de humedad, temperatura y estado de salud de cultivos de Aguacate y Mango.
- **Membresías:** Gestión de niveles de membresía (Semilla, Bosque, Reserva, Legado).
- **Inicio de Sesión Persistente:** Acceso protegido mediante contraseña demo y almacenamiento de sesión local.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Variables nativas, diseño responsivo y premium)
- JavaScript Vanilla (Lógica del cliente, simulación de datos)
- Lucide Icons (Iconografía moderna)

## 📦 Despliegue en Vercel
Este proyecto está configurado para desplegarse automáticamente como una aplicación web estática de alto rendimiento en **Vercel**.
